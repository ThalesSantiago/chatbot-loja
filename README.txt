Como Executar

1# Pré-requisitos:
Python 3.8+,
Instale as dependências:
pip install customtkinter


2# Executando o Assistente:
No terminal, navegue até a pasta backend:
cd Assistente_simples/backend

Execute o assistente:
python [bot.py](http://_vscodecontentref_/5)

3# Uso:

Digite sua dúvida ou o produto que procura na interface.
Exemplos:
"Quero um celular Samsung até 2000"
"Tem notebook Dell?"
"Quais fones mais baratos?"
"Como funciona a garantia?"
"Quero comprar o iPhone 15 Pro"

Personalização

Para adicionar ou editar produtos, altere o arquivo [`produtos.json`](backend/produtos.json).,
Para incluir novas marcas reconhecidas, edite [`marcas.json`](backend/marcas.json).,
As regras de extração de informações podem ser ajustadas em [`parser.py`](backend/parser.py).,
